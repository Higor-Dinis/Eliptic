# Eliptic

## The project
Eliptic is a open-source, MIT licensed, compiled programming language. It's in very beggining, so it cannot yet compile anything effectively.

## How to build
Run:
```bash
$  git clone https://github.com/Higor-Dinis/Eliptic && cd Eliptic && make
```
